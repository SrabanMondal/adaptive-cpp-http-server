About This Sample Website

This is a working demo running entirely on my custom C++ web server. The goal is simple: show that HTML, CSS, JavaScript, and images are being served correctly.

Two-Page Structure

The site includes a Home page and an About page, both styled with a modern UI.

Static Assets

CSS, JS, and PNG image files are delivered directly from the C++ backend without any external server.

Working Links

Navigation between pages works smoothly, confirming that the server handles routing for static files.

Ready to Extend

This setup can easily be expanded into a full project — dynamic routes, APIs, or dashboards can be added next.
